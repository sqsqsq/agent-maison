- **中立流程与 SSOT 条款**仍以 `framework/skills/<n>/SKILL.md` 为准；不要在根 Skill 正文硬编码宿主细节。

- 历史演示工程里的**钱包域**示例措辞（模块名、卡种术语等）仅作本 profile 说明；其它 `project_profile` 以各自目录下的 addendum 为准。
