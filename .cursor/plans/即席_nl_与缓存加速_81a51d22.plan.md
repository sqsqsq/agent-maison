## CLI 行为（实施后）

### Derive（`--steps`，无 `--plan`/`--steps-file`）

```bash
npm run derive-adhoc-hylyre-hint -- --bundle <id> --steps "…"
# 或
npm run adhoc-device-test -- --bundle <id> --steps "…"
```

- stdout：schema-4 JSON（**无** `planned_steps`）
- stderr：`ADHOC_DERIVE_FILE=<project>/doc/features/_adhoc/testing/reports/derive-adhoc-last.json`
- **exit 0，不调用** ensure/run

### Execute

```bash
npm run adhoc-device-test -- --bundle <id> --plan path/to/test-plan.hylyre.md
npm run adhoc-device-test -- --bundle <id> --steps-file path/to/test-steps.json
```

- lint plan md 或 `validatePlannedStepsArray(steps-file)`
- ensure → resolve → warmup（`--accept-cold-start` → 跳过 warmup）→ run
- stderr：`ADHOC_CACHE_DIR`、`ADHOC_CACHE_UPDATED`（execute 模式）
